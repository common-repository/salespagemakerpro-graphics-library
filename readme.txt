== Description ==

A premium collection of graphics and icons with tinymce based interface. The current version includes 300+ graphic items with more to be added in the future with regular updates.

== Installation ==

Due to the inbuild graphics, the package size is fairly big. This may cause some issues with the automatic install offered by the wordpress. if the inbuild installer fails, Please use the guidelines below to install the plugin.

1) Unzip salespagemakerpro-graphics-library.zip
2) Upload the salespagemakerpro-graphics-library directory to /wp-content/plugins/
3) Activate the plugin through the 'Plugins' menu in WordPress

Once activated, you can use the graphics library using the tinymce editor interface.

== Frequently Asked Questions ==

=No graphics appear in the popup =

Make sure the graphics library cache directory (wp-content/plugins/salespagemakerpro-graphics-library/cache) is having write access (chmod to 777 or 775).
